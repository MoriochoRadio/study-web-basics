## 💡 Google Analytics 추적 코드

모든 HTML 파일 상단에 본인 명의로 발급받은 Google Analytics 추적 코드(`UA-157599457-1`)가 박혀 있습니다.

```html
<script async src="https://www.googletagmanager.com/gtag/js?id=UA-157599457-1"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'UA-157599457-1');
</script>
```

웹 입문 단계에서 *"내 페이지에 누가 오는지 보고 싶다"* 는 운영자 마인드
